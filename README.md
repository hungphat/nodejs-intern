# Install nvm for MACOS
Install via Homebrew

```brew install nvm```

# Install nodejs 
Install the latest node version with: 

```nvm install node```

Use the latest node verison with 

```nvm use node```

# Check version:

```node --version```

# Install yarn

```brew install yarn```

# Which yarn you can install some_packages
Install Package:
    
```yarn add express
   yarn add body-parser
   yarn add nodemon
```
